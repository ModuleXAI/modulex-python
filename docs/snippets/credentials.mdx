## List Credentials

Retrieve all credentials for the organization, grouped by integration.

<ParamField query="integration_name" type="string" optional>
  Return a flat list for a specific integration only (e.g., `"openai"`, `"github"`).
</ParamField>

<ParamField query="auth_type" type="string" optional>
  Filter by auth type: `"api_key"`, `"bearer_token"`, `"modulex_key"`, `"custom"`.
</ParamField>

<ParamField query="limit" type="integer" optional>
  Maximum results to return (1–500). Default: `100`.
</ParamField>

```python
# All credentials, grouped by integration
result = await client.credentials.list()
print(f"Total credentials:   {result['total_credentials']}")
print(f"Total integrations:  {result['total_integrations']}")

for name, group in result["integrations"].items():
    print(f"\n{name} ({group['integration_type']}) — {group['total_count']} credentials")
    for cred in group["credentials"]:
        default = " [default]" if cred.get("is_default") else ""
        print(f"  {cred['display_name']}{default} — {cred['auth_type']}")

# Flat list for a specific integration
openai_creds = await client.credentials.list(integration_name="openai")
for cred in openai_creds["credentials"]:
    print(cred["credential_id"])
```

## Get Credential

<ParamField path="credential_id" type="string" required>
  UUID of the credential to retrieve.
</ParamField>

<ParamField query="include_masked" type="boolean" optional>
  Include partially-masked sensitive fields in the response. Default: `False`.
</ParamField>

```python
from modulex import NotFoundError

try:
    cred = await client.credentials.get(
        "cred-uuid-here",
        include_masked=True,
    )
    print(f"Integration: {cred['integration_name']}")
    print(f"Auth type:   {cred['auth_type']}")
    print(f"Default:     {cred['is_default']}")
    print(f"Last used:   {cred.get('last_used_at', 'Never')}")
    print(f"Expires:     {cred.get('expires_at', 'Never')}")
except NotFoundError:
    print("Credential not found")
```

## Create Credential

### API Key

```python
cred = await client.credentials.create(
    integration_name="openai",
    auth_data={"api_key": "sk-proj-..."},
    display_name="OpenAI Production Key",
    make_default=True,
)
print(f"Created: {cred['credential_id']}")
```

### Bearer Token

```python
cred = await client.credentials.create(
    integration_name="github",
    auth_data={"token": "ghp_xxxxxxxxxxxxxxxxxxxx"},
    auth_type="bearer_token",
    display_name="GitHub PAT",
    expires_at="2027-06-01T00:00:00Z",
    make_default=False,
)
```

### ModuleX-Managed Key

Use a key managed by ModuleX on your behalf — no credentials required.

```python
cred = await client.credentials.create(
    integration_name="openai",
    auth_type="modulex_key",
    display_name="ModuleX OpenAI Pool",
    make_default=False,
)
print(f"ModuleX key: {cred['credential_id']}")
```

### Custom Auth

```python
cred = await client.credentials.create(
    integration_name="internal_erp",
    auth_type="custom",
    auth_data={
        "api_key": "erp-key-123",
        "tenant_id": "acme-corp",
        "region": "eu-west-1",
    },
    display_name="Internal ERP System",
)
```

### MCP Server

Register an MCP (Model Context Protocol) server by URL. ModuleX will discover available tools automatically.

```python
cred = await client.credentials.create_mcp_server(
    server_url="https://mcp.acme.com/api",
    headers={"Authorization": "Bearer mcp-token-here"},
    display_name="Acme MCP Server",
)
print(f"MCP credential ID: {cred['credential_id']}")
print(f"Discovered tools: {cred.get('tools', [])}")
```

## Update Credential

Update the display name or metadata of an existing credential.

<ParamField path="credential_id" type="string" required>
  UUID of the credential to update.
</ParamField>

```python
updated = await client.credentials.update(
    "cred-uuid-here",
    display_name="OpenAI Key — Updated",
    metadata={"environment": "production", "team": "ml-platform"},
)
print(f"Updated at: {updated['updated_at']}")
```

## Delete Credential

<ParamField path="credential_id" type="string" required>
  UUID of the credential to delete.
</ParamField>

<Warning>
  Deleting a credential may break workflows and schedules that depend on it. Review usage before deleting.
</Warning>

```python
from modulex import NotFoundError

try:
    await client.credentials.delete("cred-uuid-here")
    print("Credential deleted")
except NotFoundError:
    print("Credential not found")
```

## Set Default Credential

Mark a credential as the default for its integration. Any existing default for the same integration is automatically unset.

```python
result = await client.credentials.set_default("cred-uuid-here")
print(f"Now default: {result['is_default']}")
```

## Test Temporary Credential

Validate credentials before saving them permanently.

<ParamField body="integration_name" type="string" required>
  Integration to test (e.g., `"openai"`, `"github"`).
</ParamField>

<ParamField body="auth_type" type="string" required>
  Auth type: `"api_key"`, `"bearer_token"`, or `"custom"`.
</ParamField>

<ParamField body="auth_data" type="object" required>
  The credential data to test (same shape as `create()` `auth_data`).
</ParamField>

```python
result = await client.credentials.test_temporary(
    integration_name="openai",
    auth_type="api_key",
    auth_data={"api_key": "sk-proj-..."},
)
print(f"Valid:       {result['is_valid']}")
print(f"Status code: {result['status_code']}")
print(f"Cost level:  {result.get('cost_level')}")
```

## Credential Usage

Retrieve usage statistics for a credential over a date range.

<ParamField path="credential_id" type="string" required>
  UUID of the credential.
</ParamField>

<ParamField query="start_date" type="string" optional>
  Start of the date range (ISO 8601).
</ParamField>

<ParamField query="end_date" type="string" optional>
  End of the date range (ISO 8601).
</ParamField>

```python
usage = await client.credentials.usage(
    "cred-uuid-here",
    start_date="2026-01-01T00:00:00Z",
    end_date="2026-03-01T00:00:00Z",
)
print(f"Total calls:    {usage['total_calls']}")
print(f"Successful:     {usage['successful_calls']}")
print(f"Failed:         {usage['failed_calls']}")
print(f"Success rate:   {usage['success_rate'] * 100:.1f}%")
for action, count in usage.get("action_breakdown", {}).items():
    print(f"  {action}: {count}")
```

## Credential Audit Log

Retrieve the audit log for a credential (who used it, when, and from which workflow).

<ParamField query="limit" type="integer" optional>
  Number of log entries (1–500). Default: `100`.
</ParamField>

```python
audit = await client.credentials.audit("cred-uuid-here", limit=50)
for entry in audit["entries"]:
    print(f"{entry['timestamp']} — {entry.get('action')} by {entry.get('user_id')}")
```

## Refresh MCP Tool Discovery

Re-scan an MCP server for newly added or removed tools.

```python
result = await client.credentials.refresh_mcp_discovery("cred-uuid-here")
print(f"Refreshed at:  {result['refreshed_at']}")
print(f"Total tools:   {result['total_tools']}")
print(f"Changes:       {result.get('changes', {})}")
```

## List MCP Tools

Retrieve all tools discovered from a registered MCP server.

```python
tools = await client.credentials.mcp_tools("cred-uuid-here")
print(f"Total tools: {tools['total_count']}")
for tool in tools["tools"]:
    print(f"  {tool['name']} — {tool.get('description', '')}")
```
